About My Top Movies

What is this project about?

My Top Movies is designed for movie lovers who want to organize and rate their favorite films. Users can search for movies, add them to their personal top list, rate, review, and manage them easily.

Technologies Used

  • Python (Flask) - Backend development
  • SQLite - Database for storing users and movie lists
  • Bootstrap 5 - Responsive and modern design
  • Flask-Login - User authentication system
  • TMDB API - Fetching movie data

Data Source

All movie data, including images, descriptions, and ratings, are provided by TMDB API. Personal educational project, not affiliated with TMDB.
